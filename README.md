# 📝 Python Tkinter Notepad

A lightweight **desktop Notepad application** built using **Python Tkinter**.  
This application allows users to **write, open, edit, clear, and save text files** through a simple and user-friendly GUI. The application features a clean, hacker-style interface (black background with green text) and provides standard text editing functionalities.

---

## 🚀 Features

* **Open:** Browse and open existing `.txt` files from your computer.
* **Save:** Save your current written content to a local `.txt` file.
* **Clear:** Instantly clear the entire workspace with a single click.
* **Retro UI:** Custom terminal-style text box with a black background, green foreground, and violet insert cursor.

---

## 🧠 Tech Stack

- Python 3.10+
- Tkinter (GUI)

---

## 📦 Installation & Run

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Atulframez/Python-Tkinter-Notepad.git
cd Python-Tkinter-Notepad
```

### 2️⃣ Run the Application
```bash
python main.py
```
*(Tkinter comes pre-installed with Python, so no extra setup is required.)*

## ▶️ Usage

* **Save** → Save the current text as a .txt file
* **Open** → Open an existing .txt file
* **Clear** → Clear all text from the editor
* Start typing directly in the text area
